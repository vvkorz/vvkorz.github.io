Attendance system to track home office hours

RFID based attendance systems, build in times of COVID-19 isolation to track my working hours.

March 20, 2020

  • projects
  • Raspberry Pi

Here is an attendance system that I build in times of COVID-19 isolation in home office. I needed a way to track my working hours. It uses Raspberry Pi, RFID antenna and my private Dropbox to save a csv with my hours.

The code can be found on the github

The work is inspired by this post and this post.
